## 🧠 Conceptos de POO Aplicados
En este proyecto hemos implementado los pilares fundamentales de la POO:
1. **Abstracción:** Definición de clases base y modelos de datos.
2. **Encapsulamiento:** Uso de modificadores de acceso (`private`, `public`) y métodos Getter/Setter.
3. **Herencia:** Reutilización de código mediante clases padre e hijas.
4. **Polimorfismo:** Sobrecarga y anulación de métodos para comportamientos dinámicos.
